## ğŸ”§ Setup & Installation
1. **Clone the Repository**
   ```sh
   git clone https://github.com/1Shibam/quick_chat.git
   cd quick_chat
   ```
2. **Install Dependencies**
   ```sh
   flutter pub get
   ```
3. **Set Up Firebase**
   - Create a Firebase project in the [Firebase Console](https://console.firebase.google.com/)
   - Enable Firestore and Authentication (Email/Google sign-in)
   - Download `google-services.json` (Android) and `GoogleService-Info.plist` (iOS) into respective `android/app` and `ios/Runner` folders.

4. **Run the App**
   ```sh
   flutter run
   ```
